Mars Rover

This project was done in conjuction with a capstone team and their mars rover project. I was there to help them out primarily with just the motors, but ultimately that ended up with me helping them with the majority of the digital side of the project.

Using PWM modulation we are able to control the speed of the motors, and due to an HC12 module we can control the rover from a distance using a simple Logitech F310 controller.

Blog Page

This project was essentially a partially guided lab from Dr. Angela Yu's 100 days of coding in Python course. The project was to create a blog page that included the ability to view, add, edit, and comment on posts. This was done through a couple databses that kept user and post information

Users would authenticate and then be able to view the posts, and if they were the author of the post they could edit or delete it. They could also comment on any post they wanted to. All of this info was stored in an SQLite relational database.

LED Pong

This was a final project for my digital circuits class, and was a version of pong that you could play on an LED strip.

To create this, I used an FPGA and an ADAFruit LED light strip. I learned a lot about data transmission, working with hardware, and it's where I first got my my love for microcontrollers and similar devices.

Zillow Scraper

This project was another one of the guided lab from Dr. Angela Yu's 100 days of coding in Python course. The project was to create a Python script that would scrape a Zillow page for the basic information and then put that information into a google sheet.

The increase the use I decided to use the direct Google Sheets API rather than the API Sheetly. This allowed me to have more hands on experience with the interacting with Google products.
